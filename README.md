# React Nice Resume

This is a clone from [this project](https://github.com/nordicgiant2/react-nice-resume) because i found it badly coded so i tried to make it better. Feel free to do a PR if you find something badly coded

This [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Getting Started

```bash
git clone https://github.com/OzakIOne/react-nice-resume
cd react-nice-resume/
npm i
npm run dev
```

Click this link [http://localhost:3000](http://localhost:3000)
